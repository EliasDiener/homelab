# Home Lab Server

This repository documents my personal home lab used for learning Linux, self-hosting and experimenting with AI.

## Hardware

### Home Server
- Old laptop (2013)
- Power usage: ~10–20W
- Running Ubuntu Server

### Main PC
- Desktop PC with a NVIDIA GPU
- Used for local AI experiments

## Operating Systems

- Ubuntu Server (home server)
- Ubuntu / Linux dual boot on main PC

## Services (Home Server)

- Jellyfin media server
- Nextcloud personal cloud
- CUPS printer setup

## AI Experiments (Desktop PC)

- Ollama
- Open WebUI

## Goals

- Learn Linux server administration
- Experiment with self-hosted services
- Learn C++ and game development
- Explore local AI models
